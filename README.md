# 6_latest_stories
application which will return me the latest 6 stories on time.com       


use http://&lt;localhost>/getTimeStories as a simple GET call on your browser it will provide  a simple json object array with 6 latest stories. 

Also use Pretty-print toggle on top browser page for better view.
